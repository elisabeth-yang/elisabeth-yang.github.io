Vocabulary

Access Modifiers:

private: an access modifier, ca only be accessed within this class. Some are inherently private.
public: programmer can observe this class in other places.

structures:

float: is added when it requires decimals
int: is added when it only requires integers.

Terminology and specific codes:

verticalInput = Input.GetAxis("Vertical"): is used to only work when the left or right key is pressed, however, it keyboard inputs can be changed due to the programmer's preference.
horizontalInput = Input.GetAxis("Horizontal"): used to only work when the up or down key is pressed. If there is no input the function/program goes forever.
Physics.gravity *= gravityModifier is the same as Physics.gravity = Physics.gravity * gravityModifier.
&& is the C# version of "and"
|| is the C# version of "or"
Instantiate is another way to say creating an object -- creating copies of objects that already exist.
void tells the computor that a specific function or method does not return anything, it just does what the programmers tell it to do inside the block of code.
Prefabs are prefabricated version of the object that one's using. In other words, the object can be reused, while the function of the objectt stays the same for each prefab.
